# House Price Prediction using Linear Regression

## Project Overview

This project aims to predict house prices using a **Linear Regression** machine learning model. By analyzing different housing characteristics such as area, number of bedrooms, bathrooms, parking availability, and furnishing status, the model estimates the expected price of a house. The project demonstrates how regression techniques can be applied to real estate data to build predictive models.

## Dataset

**Housing.csv**

The dataset includes several attributes related to residential properties, including property size, number of rooms, facilities, and other features that may influence housing prices.

## Objective

The main objective of this project is to develop a predictive model that can estimate house prices based on various property features. The project also aims to understand how different variables contribute to price variations.

## Steps Performed

The following steps were carried out during the project:

1. Loaded and explored the dataset to understand its structure
2. Cleaned the dataset and handled any missing values
3. Converted categorical variables into numerical form using encoding techniques
4. Selected relevant features for model training
5. Split the dataset into training and testing sets
6. Trained a **Linear Regression** model
7. Evaluated the model using performance metrics
8. Visualized the relationship between actual and predicted house prices

## Tools & Technologies Used

## Outcome

The Linear Regression model successfully identified relationships between housing attributes and property prices. The evaluation metrics indicated that the model was able to provide reasonable predictions based on the available features.

## Conclusion

This project demonstrates the process of building a regression-based machine learning model for predicting house prices. It highlights important steps such as data preprocessing, feature selection, model training, and evaluation, which are essential in real-world data science projects.
